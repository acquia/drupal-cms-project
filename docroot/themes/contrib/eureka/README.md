# Eureka Theme
This is the default theme for the Archimedes site template, developed by [Zoocha Ltd](https://www.zoocha.com). This project is a dependency of Archimedes and it is not recommended to use this theme on its own. For customisation, see CUSTOMISING.md 

## Requirements
- Drupal 11 or higher
- [Emulsify Tools](https://www.drupal.org/project/emulsify_tools) module

## Getting help
If you have trouble or questions, please visit the [issue queue](https://www.drupal.org/project/issues/eureka?categories=All) or find us on Drupal Slack.
