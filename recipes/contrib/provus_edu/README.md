# Provus®EDU — Drupal Recipe

A complete Drupal CMS recipe for higher education websites. Installs content
types, views, menus, and configuration for a production-ready college or
university site in a single command.

Built by [Promet Source](https://www.prometsource.com) — Drupal specialists
for higher education and the public sector.

## What's included
- Program & Course content types
- Faculty Directory
- News & Events
- Homepage layout (hero, program teaser, news feed, CTA)
- Student Testimonials
- Full integration with provus_edu_theme

## Requirements
- Drupal ^11
- Composer
- drupal/provus_edu_theme

## Installation
composer require drupal/provus_edu
drush recipe path/to/provus_edu
